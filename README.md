# Melding Spaces Design Lab – Portfolio Website

This repository contains the source code and assets for the portfolio website of **Melding Spaces LLP**, an architecture and interior design practice based in Bengaluru.

Founded by **Sakshi Reddy and Anusha Shetty**, the practice focuses on architecture, interior design, and urban design projects that create thoughtful, functional spaces connected to people and context.

## Contents

- Project portfolio pages
- Studio information
- Images and drawings of completed and ongoing projects
- Contact information for collaboration and inquiries

## Technologies Used

- HTML
- CSS
- JavaScript
- Responsive web design

## Purpose

The website serves as a digital portfolio to showcase selected works of **Melding Spaces LLP**, including residential interiors, architectural projects, and research work.

## Usage

You may view the portfolio online once deployed via **GitHub Pages** or another hosting platform.
